# RailsParseFixes

Applies parse fixes related to vulnerability: http://www.kb.cert.org/vuls/id/380039

## Installation

Add this line to your application's Gemfile:

    gem 'rails_parse_fixes'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rails_parse_fixes

## Usage

In an initializer, call:

    require 'rails_parse_fixes'
